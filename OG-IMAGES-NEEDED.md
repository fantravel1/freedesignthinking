# Open Graph Images Required for Social Sharing

All HTML pages reference OG (Open Graph) images for social media sharing. These images need to be created and placed in the `/images/` directory.

## Image Specifications

**Required Size:** 1200x630 pixels
**Format:** JPG (recommended) or PNG
**Aspect Ratio:** 1.91:1
**File Size:** Under 1MB (ideally under 300KB)
**Color Mode:** RGB

## Design Guidelines

### Branding Elements
- **Font Family:** 'Space Grotesk' for headings, 'Inter' for body
- **Primary Brand Color:** Gold (#d4af37)
- **Background:** Dark gradients (matching website: #1a1a2e, #16213e, #0f0f23)
- **Text Color:** Light (#e8e8e8)

### Content to Include
- Page title/theme
- "Free Design Thinking Hub" branding
- Video count or key metric
- Relevant emoji or icon
- Gradient background matching phase colors

### Best Practices
- Keep text large and readable (minimum 60px for headlines)
- Ensure high contrast between text and background
- Leave safe margins (100px on all sides)
- Test how it looks at smaller sizes (Facebook, Twitter previews)
- Include visual interest but don't overcrowd
- Maintain consistent style across all images
